# Минимум действий: «зашёл — и сразу видно кто сколько сделал»

Дашборд **только читает** таблицы — не меняет и не редактирует их. Достаточно права **«Читатель»**.

Делаешь **один раз**. Потом просто запускаешь дашборд.

---

## Шаг 1. Создать ключ и положить в проект

1. [Google Cloud Console → Credentials](https://console.cloud.google.com/apis/credentials) (выбери свой проект).
2. **Create Credentials** → **Service account** → имя любое (например `dashboard`) → **Done**.
3. Клик по созданному аккаунту → вкладка **Keys** → **Add Key** → **Create new key** → **JSON** → **Create**. Скачается файл.
4. Переименуй его в **`service_account.json`** и положи в папку **`c:\project\anchor_checker`** (рядом с `run_dashboard.py`).

**Узнать email для шага 2:** в PowerShell из `c:\project\anchor_checker` выполни:
```powershell
python show_share_email.py
```
Скопируй выведенный email.

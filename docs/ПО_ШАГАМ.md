# Дашборд — по шагам

**Подробная инструкция «как для школьника» (все твои шаги по шагам):** [ТВОИ_ШАГИ.md](ТВОИ_ШАГИ.md)

Ниже — краткий список. Делай по порядку.

---

## Часть 1. Первый запуск дашборда

**Шаг 1.** Нажми клавишу **Windows** на клавиатуре (или кнопку «Пуск»).

**Шаг 2.** Напечатай **PowerShell** и нажми **Enter** по пункту «Windows PowerShell». Откроется синее или чёрное окно.

**Шаг 3.** Введи команду и нажми **Enter**:
```
cd c:\project\anchor_checker
```
Должно появиться `PS c:\project\anchor_checker>`.

**Шаг 4.** Введи команду и нажми **Enter**:
```
python -m pip install -r requirements.txt
```
Дождись окончания (снова появится строка с `PS c:\project\anchor_checker>`).

**Шаг 5.** Введи команду и нажми **Enter**:
```
python run_dashboard.py
```
В окне появится текст с адресом, например: **Local URL: http://localhost:8501**.

**Шаг 6.** Открой браузер и в адресной строке введи: **http://localhost:8501** (или порт, который показало окно). Нажми **Enter**.

Дашборд откроется. Если данных нет — переходи к **Части 2**.

## Часть 2. Один раз настроить данные (чтобы сразу видеть «кто сколько сделал»)

Дашборд **только читает** данные из таблиц — ничего в них не меняет и не редактирует. Нужен доступ **«Читатель»**, не «Редактор».

**Шаг 7.** Открой в браузере: https://console.cloud.google.com/apis/credentials  
Выбери свой проект (например, «Dashboard META») вверху страницы.

**Шаг 8.** Нажми **+ Create Credentials** → выбери **Service account**.

**Шаг 9.** В поле «Service account name» введи, например, **dashboard** → нажми **Create and Continue** → **Done**.

**Шаг 10.** В списке кликни по только что созданному сервисному аккаунту (например, `dashboard`).

**Шаг 11.** Перейди на вкладку **Keys** → нажми **Add Key** → **Create new key** → выбери **JSON** → **Create**.  
Файл скачается на компьютер.

**Шаг 12.** Переименуй скачанный файл в **service_account.json**.  
Положи его в папку **c:\project\anchor_checker** (туда же, где лежит `run_dashboard.py`).

**Шаг 13.** В PowerShell (из папки `c:\project\anchor_checker`) введи и нажми **Enter**:
```
python show_share_email.py
```
Скопируй **email**, который появится в окне (он вида `...@....iam.gserviceaccount.com`).

**Шаг 14.** Открой первую таблицу:  
https://docs.google.com/spreadsheets/d/1DaiRFqU2d_85cXr0fDmyhzIY4V9fm0zxh4KraZMOFnw/edit  
Нажми **«Поделиться»** (справа вверху) → вставь скопированный email → выбери право **«Читатель»** → **Готово**.

**Шаг 15.** Открой вторую таблицу:  
https://docs.google.com/spreadsheets/d/1v_cheF0k0UCl9CniUWu-pTTYbl0CIOjE0BH0rzW3HXE/edit  
Снова **«Поделиться»** → тот же email → **«Читатель»** → **Готово**.

**Шаг 16.** Открой третью таблицу:  
https://docs.google.com/spreadsheets/d/1S5lk-ya4iWwq5znY_vebAuTqloyTlWTcsNuXydZXT00/edit  
**«Поделиться»** → тот же email → **«Читатель»** → **Готово**.

**Шаг 17.** Открой четвёртую таблицу:  
https://docs.google.com/spreadsheets/d/1yj3eWqTpjxZFU0e9yg5A6TIh79N-s9g1Zmz3U-Q_54E/edit  
**«Поделиться»** → тот же email → **«Читатель»** → **Готово**.

**Шаг 18.** В браузере обнови страницу дашборда (кнопка **F5** или «Обновить»). Данные должны подгрузиться — будет видно, кто и сколько разместил за выбранный период.

## Каждый раз, когда нужен дашборд

1. Открой **PowerShell**.
2. Введи: **cd c:\project\anchor_checker** → **Enter**.
3. Введи: **python run_dashboard.py** → **Enter**.
4. Открой в браузере **http://localhost:8501**.

Окно PowerShell не закрывай, пока смотришь дашборд. Чтобы выключить — в PowerShell нажми **Ctrl+C**.

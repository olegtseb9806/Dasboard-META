# Что сделать тебе, чтобы дашборд был по ссылке для всех

Всё остальное уже подготовлено (скрипты, код, инструкции). Нужны только эти шаги — их нельзя сделать без тебя.

---

## 1. GitHub: создать репозиторий и залить код

- Зайди на **https://github.com** (зарегистрируйся, если нет аккаунта).
- **New repository** → имя, например **anchor_checker** → **Create repository**.
- В PowerShell выполни по очереди (подставь вместо `ТВОЙ_ЛОГИН` свой логин GitHub):

```powershell
cd c:\project\anchor_checker
git init
git add .
git status
```
Проверь: в списке **не должно** быть `service_account.json` и `service_account.json.json`. Если они есть — не делай коммит, напиши в чат.

```powershell
git commit -m "Dashboard"
git remote add origin https://github.com/ТВОЙ_ЛОГИН/anchor_checker.git
git branch -M main
git push -u origin main
```
Если попросит авторизацию — логин GitHub, пароль: **Personal Access Token** (GitHub → Settings → Developer settings → Personal access tokens → Generate new token, право **repo**).

---

## 2. Streamlit Cloud: создать приложение

- Открой **https://share.streamlit.io**, войди через **GitHub**.
- **New app** → Repository: **твой_логин/anchor_checker**, Branch: **main**, Main file path: **app/dashboard/app.py** → **Deploy**.
- Дождись окончания деплоя (1–3 минуты).

---

## 3. Вставить секрет с ключом

- В PowerShell из папки `c:\project\anchor_checker` выполни:
```powershell
python prepare_streamlit_secret.py
```
- **Скопируй весь вывод** (от `service_account_json` до последней кавычки `'''`).
- В Streamlit Cloud открой своё приложение → **Settings** (шестерёнка) → **Secrets** → вставь скопированное → **Save**.

Через полминуты обнови страницу дашборда — должны подтянуться данные.

---

## 4. Раздать ссылку

- Вверху страницы приложения на Streamlit Cloud скопируй ссылку вида **https://....streamlit.app**.
- Отправь её сотрудникам (чат, почта, закладки). Кто открывает — видит дашборд, без установки и твоего компьютера.
